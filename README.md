# Pneumonia Detection System

## Description
This project uses deep learning techniques to detect pneumonia from chest X-ray images.

## Technologies Used
- Python
- Deep Learning
- CNN
- TensorFlow / Keras

## Dataset
Chest X-ray images (Kaggle dataset)
